# Changelog

**05/20/2026 -- The release workflow was updated so IH-Depth files unpack directly into the raw IH scene folders beside the matching `.hdr/.bsq` files. Depth labels are public `uint16` PNG files, and the repository now focuses on dataset setup, split preparation, and evaluation. Physics-based baseline reproduction files were removed from the tracked release branch for now to avoid mixing calibration assets and baseline-specific files with the dataset usability release.**

**04/18/2026 -- The repository was cleaned up to remove dataset-construction artifacts and narrow the public surface to instructions, benchmarking, and baselines.**
